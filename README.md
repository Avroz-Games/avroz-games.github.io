# Avroz Tecnologia — FinOps Multi-Cloud

Site institucional da Avroz Tecnologia: consultoria FinOps e migração de datacenters para AWS, Azure, Google Cloud e Oracle.

## Desenvolvimento local

```bash
npm install
npm run dev
```

Acesse [http://localhost:5173](http://localhost:5173).

## Build e deploy

O deploy para GitHub Pages ocorre automaticamente via GitHub Actions ao fazer push na branch `main`.

```bash
npm run build
npm run preview
```
